1) What is the difference between `var`, `let`, and `const`?

`var`, `let`, and `const` are all used to store data in JavaScript, but they work in different ways.

- `var` is the old way of declaring variables in JavaScript. It can be changed later, and it has some confusing behavior because of function scope and hoisting.
- `let` is the modern way to declare a variable when the value may change later.
- `const` is also a modern way to declare a variable, but it is used when the value should not be reassigned.

map() = change every item and return a new array

5) What are template literals?

Template literals are a modern way to write strings in JavaScript using backticks instead of quotes.
